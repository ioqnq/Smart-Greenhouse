# üå± Smart-Greenhouse
A smart greenhouse IOT project consisting of a watering and fanning system done with MicroPython on a Rasperry Pi Pico 2W. Control soil humidity and temperature through a Flutter app which uses Firebase authentification and storage to provide remote user access and environmental factor analytics.
